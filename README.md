# Benedict Clinic Theme - WordPress Theme

WordPress тема для урологічної клініки Benedict Clinic. Сучасний landing page з повною адаптивністю та всіма необхідними функціями для презентації продукту.

## Особливості

- ✅ Адаптивний дизайн (мобільні, планшети, десктопи)
- ✅ Hero секція з відео на фоні
- ✅ Блок опису продукту та переваг
- ✅ Галерея застосування
- ✅ Реалізовані кейси (до/після)
- ✅ Відгуки клієнтів
- ✅ Контактна форма
- ✅ Інтеграція Contact Form 7
- ✅ Плавна прокрутка та навігація
- ✅ Мобільне меню
- ✅ SEO-оптимізована розмітка

## Встановлення

### 1. Активувати тему

1. Увійдіть в адмін-панель WordPress
2. Перейдіть до **Зовнішній вигляд → Темы**
3. Знайдіть тему "Benedict Clinic Theme"
4. Натисніть **Активувати**

### 2. Активувати Contact Form 7

1. Перейдіть до **Плагіни**
2. Знайдіть плагін "Contact Form 7"
3. Натисніть **Активувати**

### 3. Налаштування форм

Після активації Contact Form 7:

1. Перейдіть до **Contact → Contact Forms**
2. Створіть нову форму або відредагуйте існуючу
3. Використовуйте наступний код форми:

```html
<label>Ім'я*
[text* name]</label>

<label>Телефон*
[tel* phone]</label>

<label>Коментар (необов'язково)
[textarea comment]</label>

[submit "Відправити"]
```

4. Збережіть форму та скопіюйте shortcode (наприклад, `[contact-form-7 id="1"]`)
5. Shortcode вже включений в шаблони головної та контактної сторінки

## Використання

### Головна сторінка

Головна сторінка автоматично використовує шаблон `front-page.php` і містить:
- Hero секцію з відео
- Опис продукту та переваги
- Галерею застосування
- Реалізовані кейси
- Контактну форму

### Створення нових сторінок

1. Перейдіть до **Сторінки → Додати нову**
2. Оберіть потрібний шаблон зі списку "Template"
3. Заповніть контент

## Технічні вимоги

- WordPress 5.0 або новіший
- PHP 7.4 або новіший
- Contact Form 7 (рекомендовано)
